Technologies Used

These are the technologies I commonly use or had long experience with for development since the starting my coding journey.

Python

Python is the first programming language I have used. It is a dynamically typed language famous for its ease-of-use, making it the programming language I am most comfortable with.

JavaScript

JavaScript is the second programming language I used. Similar to Python, it is also dynamically typed and is fairly easy to use, helping me make interactive and functional websites.

TypeScript

TypeScript is a superset of JavaScript, having additional features like static typing. The tooling and ecosystem of TypeScript significantly improves its developer experience.

Go

Go is currently my favourite programming language due to its great built-in libraries like the HTTP library, as well as its great performance.

Cloudflare

Cloudflare is a cloud platform that offers various services like Cloudflare Registrar. Cloudflare Pages is my go-to way to host serverless websites for free.

Docker

Docker is a tool that allows software to be containerised, ensuring that the software runs consistently accross multiple devices from development to production.

Git

As someone who creates multiple copies for each version before learning Git, it has made developing more convinient especially in a groupwork.

Arch Linux

I do not usually say this but, I use Arch btw. Linux in general is a great operating system especially for development due to the CLI tools built on it.

Neovim

Neovim is a text editor in the terminal that have a great ecosystem. It has helped me enjoy coding 10x more and made me code faster.
